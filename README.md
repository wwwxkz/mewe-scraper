### Usage

Cookies needed:

- refresh-token
- access-token

Ways to get these cookies:

(Firefox)
- Cookie Quick Manager 
(https://addons.mozilla.org/pt-BR/firefox/addon/cookie-quick-manager/)

(Chromium)
- EditThisCookie
(https://chrome.google.com/webstore/detail/editthiscookie/fngmhnnpilhplaeedifhccceomclgfbg)
